# MSLAN
 🔥 This repository is the official implementation corresponding to our submission to *The Visual Computer*. Please cite our paper if you use this code.

## 📌 Overview

Crowd counting is a challenging task due to large scale variations and complex background noise.  
We propose **MSLAN**, a multi-scale spatial location awareness network based on MPViT and attention mechanisms.

Key components:
- Multi-Path Vision Transformer (MPViT-small) backbone
- Multi-scale Group-wise Attention (MGA)
- Spatial Location Awareness Module (SLAM)
